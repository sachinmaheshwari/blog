Jekyll on Heroku
----------------

[What is this?](http://www.jamesward.com/2014/09/24/jekyll-on-heroku)

[![Deploy on Heroku](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)

Run locally:

- Install rbenv
- `rbenv install`
- `gem install bundler`
- `bundle install`
- `bundle exec jekyll serve --watch`
